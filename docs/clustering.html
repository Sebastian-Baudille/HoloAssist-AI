Pipeline

Clustering Pipeline

The clustering pipeline transforms raw point clouds into 3D cube centroids suitable for the RL observation vector.

01

Transform

Camera body frame to world frame using the static TF chain.

›
02

Height Crop

Z-crop above table surface: z_min = table_top + 0.015, z_max = table_top + cube_height + 0.01

›
03

DBSCAN

Density-based clustering on XYZ only. No prior knowledge of cube count needed. color_weight = 0.0

›
04

Centroids

Per-cluster mean yields world-frame (x, y, z) centroids fed to the PPO state vector.

Key Parameters

ParameterValueDescription
z_mintable_top + 0.0153× sensor noise margin
z_maxtable_top + cube_height + 0.01Upper crop bound
color_weight0.0XYZ-only clustering
centroidcluster_pts.mean(axis=0)World-frame mean per cluster

Validation Results

Validated across 60 scenes with varying cube configurations:

  • Centroid error: ~1.6 cm vs ground truth (within sensor noise σ = 0.005 m)
  • Cube recall: 100% — all cubes detected in every scene
  • Output shape: (k*3,) = (12,) for 4 cubes

Visualisation

Polyscope renders the cropped point cloud with cluster colours and centroid markers. PCA axes are shown for orientation (visualisation only).
