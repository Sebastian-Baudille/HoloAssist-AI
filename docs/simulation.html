Assets

Robot Asset Pipeline

The UR3e + RG2 robot is modelled as a layered USD asset:

isaac_rl/assets/ ├── meshes/ 22 mesh files (14 UR3e .dae/.stl + 8 RG2 .stl) ├── urdf/ │ ├── ur_onrobot.urdf raw xacro flatten │ └── ur_onrobot_prepared.urdf mimic-stripped, URIs rewritten └── usd/ └── ur_onrobot_prepared/ layered USD ├── ur_onrobot_prepared_base.usd ├── ur_onrobot_prepared_physics.usd ├── ur_onrobot_prepared_robot.usd └── ur_onrobot_prepared_sensor.usd

URDF Preparation

prepare_urdf.py strips mimic joints and rewrites mesh URIs for Isaac Sim import. The resulting URDF is imported into USD via the Isaac Sim asset converter.

Environment

Isaac Lab Environment

The reach task uses Isaac Lab's DirectRLEnv API with modular strategy pattern:

holoassist_tasks/tasks/direct/reach/ ├── __init__.py task registration + env config ├── reach_env.py DirectRLEnv subclass ├── reach_env_cfg.py scene, robot, action, obs configs ├── actions/ │ └── joint_delta.py per-joint scaled delta actions ├── observations/ │ └── ground_truth_12d.py joint pos + EE pos + target pos └── rewards/ ├── dense_reach.py distance-based dense reward ├── dense_reach_v1.py + convergence bonus └── dense_reach_v2.py + action penalty
